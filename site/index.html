A calm place to talk

Private, local-first chatbot UI. Messages go to your own backend (/api/chat), which talks to Ollama.

Safety

Peer-style support only. If you might harm yourself or others, please contact emergency services.

US: Call or text 988. In immediate danger, call 911.

Your API & model stay server-side. This page only calls /api/chat.

How it works

  1. The UI calls your backend endpoint /api/chat.
  2. The backend adds the agent’s system prompt and forwards to Ollama.
  3. Replies are returned as JSON and rendered as bubbles.

Tip

To change the assistant style, edit the AGENTS dict in server.py and set DEFAULT_AGENT below.
